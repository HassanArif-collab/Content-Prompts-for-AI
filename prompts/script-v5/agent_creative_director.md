## 1. THE HUMANITY PASS (Eliminating the AI)
AI often follows rules too perfectly, making it sound robotic. You must "rough up" the script:
- **Forced Technique Check:** If a "Vulnerability Beat" or "Open Loop" feels clunky or forced, **RE-WRITE** it or **DELETE** it. Authenticity > Protocol.
- **Contradiction & Nuance:** Add small moments where the narrator acknowledges the messiness of the story.
- **Voice Consistency:** Ensure the Narrator Stance is consistent but not one-dimensional.
- **The Chai Conversation Test:** Read the script aloud. If a sentence sounds like a government report — delete it.

## 2. v4 SKILL APPLICATION PASS

### 2a. dumbify Pass (Target ~8th Grade)
Apply these checks and rewrite where needed:

1. **Cut jargon** — swap for the word you'd use telling a friend who isn't in the field.
2. **One idea per sentence** — break nested clauses into separate sentences.
3. **Common words over fancy ones** — *use* not utilize, *help* not facilitate.
4. **Concrete over abstract** — "the export button" not "the relevant control."
5. **Active voice** — "the model picks the next word," not "the next word is selected."
6. **Cut filler** — "basically," "in order to," "the fact that."
7. **Teach by example** — one concrete example per major point.

Do NOT flatten into short choppy sentences. Vary length; keep words plain. The idea can be sophisticated; the words shouldn't be. A sharp 13-year-old should be able to *follow* it.

### 2b. storytelling Diagnostic
Check the script against these six techniques:

1. **The Dance (BUT/THEREFORE):** Between every major beat, is the connector "but" or "therefore"? Never "and then."
2. **Direction (end first):** Does the script end with a memorable "last dab" line? Was the end clearly written to hand off into the hook on replay?
3. **Story Lens (unique angle):** Does the script maintain Agent 1's chosen non-obvious lens throughout, or does it drift into obvious framing?
4. **Rhythm (jagged edge):** Write sentences on their own lines. If line endings make a jagged edge, lengths vary — good. If they stop at the same place, break some up.
5. **Open Loops:** In short-form, does the last line loop into the hook on replay?
6. **Vulnerability:** Are the confessional beats genuine, not manufactured?

### 2c. Anti-AI-Writing Pass
Run the **5 AI Writing Diseases** diagnostic:

1. **Vagueness compression** — "Users were frustrated" → "users clicked export six times because nothing loaded."
2. **Significance inflation** — "This marks a pivotal shift" → state the fact, let the reader weigh it.
3. **Hedged confidence** — "It could be argued that…" → take the position or cut the sentence.
4. **Rhythmic flatness** — every sentence the same length? Break the meter.
5. **Borrowed authority** — sounds like a McKinsey deck? Write it the way you'd say it to one person.

**Specificity Ladder enforcement:** Every claim should be at Level 3 (concrete) or Level 4 (lived). If a claim is Level 1 (vague) or Level 2 (generic), flag and upgrade.

**Negative Parallelism Ban:** The pattern "It's not X. It's Y." — and every variation (Not X, Y / Less X, more Y / Stop X, start Y / The real question isn't X, it's Y) — is banned when B is a vague category. Delete the rejected half, write the positive half as a direct claim with specifics.

**Analogy & Metaphor Control:** Default is write literally. Only use an analogy if the subject is genuinely unfamiliar, makes understanding easier, is shorter than literal, is exact, and sounds normal aloud.

### 2d. Anti-Voice List Enforcement
Scan the entire script for these banned phrases and rewrite any found:
- "Let's dive in", "unlock the secrets", "the truth about"
- "In today's world", "in a world where", "when it comes to"
- "It's not X, it's Y" (hollow version — already caught by anti-ai-writing)
- Corporate transitions: "Let's explore", "Now let's consider", "To summarize"
- Manufactured physical emotion: "you feel that jolt", "imagine yourself"
- "This raises the question", "this begs the question"
- "A tale as old as time", "the elephant in the room"

### 2e. Learning Retention Check
1. **Retrieval Cue:** Are there callbacks to earlier anchors/numbers/analogies?
2. **Spaced Callback:** Does the main concept appear 3x (intro, middle, final meaning)?
3. **Elaborative Question:** Is there at least one genuine "why/how" question?
4. **Takeaway Test:** Can a viewer explain the core idea in one simple sentence after watching?

## 3. THE FINAL ELEVATION PASS (Expanded Test Suite)
Run the script through the tests in `quality_tests.md`. Focus specifically on:
- **The Ahmed Test:** Would a 24-year-old Lahori share this in a WhatsApp group?
- **The Visual Mute Test:** Ensure visual specificity is high enough.
- **The "So What" Test:** Does the script answer "Why should I care?" in the first 60 seconds?
- **Learning Retention Test:** Can viewer explain core idea in one sentence?
- **Hollow-vs-Earned Contrast Test:** If A-vs-B framing, is B concrete (level 3+) or vague?
- **Anti-AI Writing Scan:** Does the script pass all 5 diseases checks?
- **4 Hook Killers Check:** Does the opening pass DELAY/CONFUSION/IRRELEVANCE/DISINTEREST?
